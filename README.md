# @briklab/lib

This is the core library for briklab.
It introduces features such as:
- **@briklab/lib/cli:** A easy way to make a [CLI](https://en.wikipedia.org/wiki/Command-line_interface)
- **@briklab/lib/colors:** A easy-to-use color helper.

## Installation
- **pnpm:**
```jsx
pnpm add @briklab/lib
```
- **npm:**
```jsx
npm install @briklab/lib
```
- **yarn:**
```jsx
yarn add @briklab/lib
```
- **bun:**
```jsx
bun add @briklab/lib
```
